## 二、复盘六问（结构化分析）
### 2.1 大盘情绪怎么样？
**情绪温度计算公式**：

```plain
情绪温度 = (涨停数/上涨数) × 连板高度系数 × (1-炸板率)

温度分级：
- 冰点（<20）  ：只做超跌反弹，空仓为主
- 回暖（20-40）：可以低吸龙头，2-3成仓
- 高温（40-60）：追涨龙头，3-5成仓
- 过热（>60）  ：谨慎，随时准备撤退，只持有已盈利票

关键指标：
- 涨停数：>50个=活跃，30-50=正常，<30=低迷
- 连板高度：≥5板=情绪高潮，3-4板=健康，≤2板=分化
- 炸板率：<30%=健康，30-50%=分化，>50%=情绪转弱
```

### 2.2 是否有逻辑？（结构化评分）
**逻辑强度评分系统（满分100分）**：

```plain
逻辑强度 = 政策催化(30分) + 事件驱动(25分) + 业绩支撑(20分) + 
          题材热度(15分) + 机构认可度(10分)

评分细则：
【政策催化 30分】
- 中央级文件/会议：30分（如中央经济工作会议）
- 部委政策/规划：20分（如工信部产业政策）
- 地方政策/试点：10分（如地方补贴政策）
- 无政策支撑：0分

【事件驱动 25分】
- 突发重大事件：25分（如地缘冲突、自然灾害）
- 预期内事件：15分（如财报、新品发布）
- 周期性事件：5分（如节假日消费）
- 无事件驱动：0分

【业绩支撑 20分】
- 业绩暴增（>100%）：20分
- 业绩稳增（30-100%）：15分
- 业绩平稳（0-30%）：5分
- 业绩下滑：0分

【题材热度 15分】
- 开盘啦热度Top3：15分
- 东财概念资金流入Top10：10分
- 近3日持续活跃：5分
- 冷门题材：0分

【机构认可度 10分】
- 近期多家机构调研/推荐：10分
- 有机构调研：5分
- 无机构关注：0分

逻辑强度判断：
- ≥70分：强逻辑，可重仓
- 50-69分：中等逻辑，轻仓
- <50分：弱逻辑，观察或放弃
```

### 2.3 分别是什么板块？是否是细分龙头？
**板块地位评估**：

```plain
龙头判断标准：
1. 板块内涨幅排名：Top3
2. 板块内资金流入：Top3
3. 板块内封板时间：最早或前3
4. 板块内成交额：Top5（有容量）
5. 板块内连板高度：最高或并列最高

细分板块龙头：
- 如果大板块龙头已经高位，细分板块龙1也可以
- 例如：新能源（大板块）→ 储能（细分）→ 某龙头股

板块生命周期判断：
- 启动期（1-3天）：龙头刚出现，最佳介入期 ⭐⭐⭐⭐⭐
- 加速期（3-5天）：板块共振，可以追强 ⭐⭐⭐⭐
- 分化期（5-7天）：只做龙头，回避跟风 ⭐⭐⭐
- 衰退期（7天+）：不碰 ⭐

实现方式：记录板块首次异动日期，自动标注所处阶段
```

### 2.4 走得有没有主动性？
**主动性量化评分系统（满分100分）**：

```plain
主动性得分 = 封板速度(30分) + 封单强度(25分) + 
            分时形态(20分) + 换手率(15分) + 
            大单占比(10分)

评分细则：
【封板速度 30分】
- 09:30-10:00封板：30分（强势）
- 10:00-11:30封板：20分（正常）
- 13:00-14:00封板：10分（偏弱）
- 14:00-15:00封板：5分（很弱）
- 尾盘封板：2分（跟风）

【封单强度 25分】
- 封单/流通市值 >5%：25分
- 封单/流通市值 3-5%：15分
- 封单/流通市值 1-3%：8分
- 封单/流通市值 <1%：3分

【分时形态 20分】
- 一波流（直线封板）：20分
- 二波流（回踩后封板）：15分
- 震荡封板（多次冲击）：8分
- 烂板（反复开板）：3分

【换手率 15分】
- 10-20%（充分换手）：15分
- 5-10%（正常）：10分
- 20-30%（偏高）：5分
- <5%或>30%：3分

【大单占比 10分】
- 大单净流入占比>60%：10分
- 大单净流入占比40-60%：6分
- 大单净流入占比20-40%：3分
- 大单净流入占比<20%：0分

主动性判断：
- ≥70分：强主动，优先买入
- 50-69分：中等主动，可以考虑
- <50分：被动跟风，放弃
```

### 2.5 大盘区间强度分析
**盘中关键时段监控**：

```plain
关键时段：
- 09:30-10:00：开盘情绪（最重要）
- 10:00-10:30：早盘分化
- 13:00-13:30：午后情绪
- 14:00-14:30：尾盘博弈

监控方法：
1. 看板块：哪个板块在这个时段启动？
2. 看个股：板块内哪只股票带动的？
3. 看联动：其他板块是否跟随？
4. 看持续：这个强度能否延续到收盘？

实时监控：
- 每15分钟看一次涨停板（新增了哪些？）
- 每30分钟看一次板块资金流入（哪个板块在加速？）
- 每小时看一次个股分时（龙头是否还在强势？）
```

### 2.6 是否唯一？（动态监控）
**唯一性评估与动态对比**：

```plain
唯一性判断：
- 同题材只有1只涨停：唯一性强 ⭐⭐⭐⭐⭐
- 同题材2只涨停：可能都要买 ⭐⭐⭐⭐
- 同题材3-5只涨停：选最强的 ⭐⭐⭐
- 同题材>5只涨停：板块共振，但要防分化 ⭐⭐

动态监控机制：
- 盘中实时对比：如果同题材出现更强标的（更早封板、更大封单），自动预警
- 次日竞价验证：如果竞价溢价率低于同题材其他股，降低评分
- 龙头切换信号：如果连续2天被同题材其他股超越，考虑换股

唯一性失效处理：
- 如果今天唯一，明天出现更强的 → 盘中换股
- 如果今天并列，明天一个走强一个走弱 → 持有强者
```

## 三、理想标的评分系统（11维度量化）
### 3.1 评分维度与权重
```plain
总分 = 核心维度(50分) + 重要维度(30分) + 辅助维度(20分)

【核心维度 50分】
1. 逻辑强度（15分）：≥70分=15分，50-69分=10分，<50分=5分
2. 主动性（15分）：≥70分=15分，50-69分=10分，<50分=5分
3. 龙头地位（20分）：
   - 板块龙1：20分
   - 细分龙1：15分
   - 板块前3：10分
   - 板块前5：5分

【重要维度 30分】
4. 唯一性（10分）：唯一=10分，2只=7分，3-5只=4分，>5只=2分
5. 带动情绪（10分）：
   - 带动板块启动：10分
   - 板块内最强：7分
   - 跟随板块：4分
   - 独立行情：2分
6. 大众情人（10分）：
   - 多个游资席位：10分
   - 单一游资+机构：7分
   - 纯游资或纯机构：4分
   - 无明显主力：0分

【辅助维度 20分】
7. 成交容量（3分）：>10亿=3分，5-10亿=2分，<5亿=1分
8. 上涨趋势（3分）：20日线上=3分，60日线上=2分，其他=0分
9. 跟风股（3分）：有3只以上=3分，1-2只=2分，无=0分
10. 未加速（3分）：未连续3天缩量=3分，已加速=0分
11. 反包能力（8分）：
    - 昨日暴量今日反包：8分
    - 昨日炸板今日封板：6分
    - 昨日缩量今日放量：4分
    - 无反包特征：0分
```

### 3.2 买入标准
```plain
评分结果：
- ≥80分：强烈买入信号，可半仓
- 70-79分：买入信号，可3成仓
- 60-69分：观察，可1成仓试探
- <60分：放弃

特殊情况：
- 如果核心维度<30分，即使总分≥80也不买（逻辑或主动性有问题）
- 如果情绪温度<20（冰点），即使总分≥80也只能1成仓试探
- 如果情绪温度>60（过热），即使总分≥80也要谨慎，优先持有已盈利票
```

## 四、仓位管理与买入策略
### 4.1 动态仓位管理
```plain
根据市场情绪温度调整仓位：

【冰点期 <20】
- 总仓位：0-1成
- 持股数：0-1只
- 策略：空仓为主，或1成仓试探超跌反弹

【回暖期 20-40】
- 总仓位：2-3成
- 持股数：1只
- 策略：低吸龙头，不追高

【高温期 40-60】
- 总仓位：3-5成
- 持股数：1-2只
- 策略：可以追涨龙头，但要设好止损

【过热期 >60】
- 总仓位：只持有已盈利票，不新开仓
- 持股数：1-2只（已持有的）
- 策略：随时准备撤退，落袋为安
```

### 4.2 买入原则
```plain
1. 最多只能买2只（多了看不过来，而且只想买最强的）
2. 单只最多半仓（仓位太重，影响心态）
3. 做完一个，再做另一个（不要同时操作多只）
4. 优先买评分最高的（≥80分）
5. 如果两只评分接近，买唯一性更强的
6. 如果仓位乱了，停止任何买入，先理清持仓
```

### 4.3 买入时机
```plain
盘中买入：
- 首板：10:00前封板，可以排板
- 2-3板：分时回踩不破均价线，可以低吸
- 4板以上：不追，等回调

次日买入：
- 竞价高开3-5%，可以挂单买入
- 竞价平开或低开，观察开盘后走势
- 开盘后快速拉升，可以追（但要设止损）
```

## 五、退出机制（止损+止盈+换股）
### 5.1 止损线（严格执行）
```plain
【当日止损】
- 炸板不回封：盘中立即止损（除非是洗盘性质的秒开秒封）
- 尾盘开板：14:30后开板，盘中止损
- 大幅开板：开板后跌幅>3%，盘中止损

【次日止损】
- 竞价低开>3%：竞价出
- 开盘后快速跳水：跌破昨日均价线，盘中止损
- 板块龙头换人：同题材出现更强标的，盘中换股

【情绪止损】
- 市场情绪转冰点（<20）：全部清仓
- 连板高度断层（5板直接到2板）：减仓一半
- 炸板率>50%：不再新开仓，持有的设好止损
```

### 5.2 止盈线（分批止盈）
```plain
【首板止盈】
- 不设止盈，看次日表现

【2-3板止盈】
- 不设止盈，看情绪和板块
- 如果板块开始分化，减半仓

【4-5板止盈】
- 减半仓，防止分化
- 如果次日继续强势，可以加回来

【6板以上止盈】
- 只留底仓（1-2成），落袋为安
- 如果冲击10板，可以留底仓博弈

【特殊止盈】
- 单日涨幅>15%（未封板）：尾盘减半仓
- 连续3天缩量加速：减半仓（加速末期）
- 暴量涨停（换手>30%）：次日冲高减半仓
```

### 5.3 换股策略
```plain
【主动换股】
- 持有的票走弱，同题材出现更强的 → 盘中换股
- 持有的票已经高位（5板以上），新龙头刚启动 → 减仓换股
- 板块生命周期进入衰退期，新板块启动 → 换板块

【被动换股】
- 持有的票炸板，同题材龙头还在强势 → 止损后换龙头
- 持有的票逻辑证伪，新逻辑出现 → 止损后换新逻辑

【换股原则】
- 不要频繁换股（一天最多换1次）
- 换股要换更强的（评分更高、主动性更强）
- 换股要在盘中完成（不要隔日换）
```

## 七、注意事项与纪律
### 7.1 铁律（必须遵守）
```plain
1. 仓位要配置好，如果乱了，停止任何买入
2. 做完一个，再做另一个（不要同时操作多只）
3. 行情不是1天走完的，但是钱可以被亏完
4. 尽量自己去亲力亲为地看（看K线、看板块、看分时）
5. 严格执行止损，不要心存侥幸
6. 不要追高，宁可错过，不要做错
7. 不要频繁交易，每天最多操作2次
8. 不要在情绪冰点期重仓，不要在情绪过热期追高
```

### 7.2 心态管理
```plain
1. 连续亏损2次，停止交易1天，复盘反思
2. 单日亏损>5%，停止交易，第二天再来
3. 单周亏损>10%，停止交易1周，重新审视策略
4. 不要因为踏空而追高，不要因为亏损而报复性交易
5. 保持冷静，不要被情绪左右
```

### 7.3 持续优化
```plain
1. 每天复盘，记录日志
2. 每周总结，优化模型
3. 每月回测，调整策略
4. 每季度审视，是否需要大调整

记住：
- 这套系统不是完美的，需要不断优化
- 市场在变化，策略也要跟着变化
- 不要迷信任何指标，要结合实际情况判断
- 最重要的是执行力和纪律性
```

## 八、快速检查清单（盘前/盘中/盘后）
### 8.1 盘前检查（09:00-09:25）
```plain
□ 查看隔夜消息（政策、事件、外盘）
□ 查看昨日龙虎榜（主力动向）
□ 查看今日集合竞价异动（高开、放量）
□ 查看板块资金流向（哪个板块在启动）
□ 计算市场情绪温度（预判今日情绪）
□ 确定今日仓位上限（根据情绪温度）
```

### 8.2 盘中检查（09:30-15:00）
```plain
□ 09:30-10:00：关注开盘情绪，哪些股票封板
□ 10:00：第一次复盘，评分>80的股票加入自选
□ 11:30：中盘复盘，板块是否持续强势
□ 13:00-13:30：关注午后情绪，是否有新龙头
□ 14:00：第二次复盘，决定是否买入
□ 14:30：尾盘观察，持仓是否需要调整
```

### 8.3 盘后检查（15:00-17:00）
```plain
□ 查看龙虎榜数据（主力席位）
□ 查看板块资金流向（全天汇总）
□ 复盘今日涨停股（评分、分类）
□ 记录复盘日志（买入、卖出、踏空）
□ 更新股票池数据库（高频标的）
□ 制定明日计划（重点关注哪些股票）
```
