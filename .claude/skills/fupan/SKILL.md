## ⚠️ 数据陷阱清单（必读，每次复盘前先自检）

血泪教训，每条都曾让用户实际亏过钱或险些亏钱：

1. **必用前复权**：AkShare K线接口默认不复权，会因除权除息严重失真。必须 `adjust="qfq"`。**典型反例**：科创板某AI芯片股某日不复权显示 -36.7%，实际仅 -5.6%（当日除权除息）。
2. **物理常识检验**：A股主板±10%、科创板/创业板/北交所±20%。**看到超过这个范围的涨跌幅立刻怀疑除权除息**。
3. **量能单位**：科学计数法 `1.5e+10 = 150亿`，不是 15亿。读取 amount 字段后做合理性检查（大蓝筹日成交几十-百亿）。
4. **概念归类陷阱**：东财行业分类会把同一概念打散（如机器人/Optimus 散落在「通用设备/专用设备/汽车零部/自动化设」4个行业）。**必须按概念归类，不能只看东财行业**，否则会漏判主线。**⚠️同时反向陷阱：一只股票常有多个概念（如pcb+ai算力+消费电子），只取第一个/只看东财行业 → 把票分到错误板块。≥2板个股必须走 HARD GATE 1.5「板块归属矩阵」**，列全所有概念逐个对比同概念热度，按"高度+密集度"判定主板块。
5. **多源验证**：单一数据源可能错（哪怕新浪+腾讯都给不复权数据时两源一致也是错）。至少用 2 源 + 物理常识 双重验证。
6. ~~**海外联动必查**~~：**已废弃**（feedback_no_overseas_linkage）。/fupan A股日复盘**禁止**写任何美股个股/海外5分组联动段落，第零步跳过海外数据获取。
7. **龙虎榜累计陷阱**：连板上榜（"连续三个交易日内累计涨幅偏离值达20%"）的"龙虎榜买入/净买入额"是 **3天累计**，不是单日。要除以天数估算单日。
8. **AkShare 网络故障转移**：`push2.*.eastmoney.com` 系列 CDN 可能不通；但 `datacenter-web.eastmoney.com`（涨停池+龙虎榜）+ 新浪K线 + 腾讯K线 通常能用。失败时主动切换数据源。
9. **多次首板 = 反复股**：3次以上首板每次都炸/失败的票，下一次大概率再失败，**不是真龙头**。
10. **机构出货明牌**：**涨停日**机构净卖出 >1亿是顶级警告（曾出现过单日机构净卖7亿+、4亿+的真实案例）— 这种票次日**绝对不能碰**。
11. **platerotat 历史回溯限制**：`platerotat.py` 对**历史日期**（>3 个交易日前）数据稀疏不可用——脚本只对当天的强度榜完整，历史日期点位多数为 `None`。**典型反例**：2026-05-11 这种 10 天前的日期，platerotat 仅返回 0-2 个板块有数据，对"板块生命周期判定"事实上不可用。**降级方案**：历史日期复盘时，板块周期判定**禁止依赖** platerotat，改用"涨停密度 + 跟风票表现 + 涨停股梯队"三块证据替代，并在报告里明确标注「platerotat 历史数据不可用」。
12. **分歧日 ≠ 破坏日**：炸板≥5次但回封了的连板，多半是真龙头的"分歧转一致"经典走势，不是衰退。**典型反例**：2026-05-08 大唐发电 3板炸11次但最终封死成3板，旧版判"补涨末端·电力衰退·排除"；5-11 4板封死+板块4只涨停爆发，整个判定方向打脸。**强制流程**：≥2板个股炸板≥5次时，必须走 HARD GATE 1.6「分歧日 5 问识别」，不能仅凭"炸板多"就贬低评分或标"盘面破坏"。

## 🎯 核心理念（V3.1）

**连板+波段+周期理论三位一体**：
- 首波龙头连板期：重仓吃连板爆发力
- 补涨龙连板期：轻仓参与，连板结束立刻清仓
- 回调期：减仓但不清仓（保持20%），等待二波连板
- 二波连板期：重仓再吃连板

**关键区别**：
- ❌ 错误：连板结束就清仓删自选 → 错过二波
- ✅ 正确：首波龙头连板结束减仓至20%，加入监控池 → 二波再连板时重仓上
- ⚠️ 重要：补涨龙连板结束必须清仓，不等二波（因为是板块末期）

**【规则6】真龙三标准——不能默认"涨停=真龙"（feedback_leader_definition）**

判定"真龙"必须用三大标准，**禁止用"是否涨停"作前提**：

1. **板块周期内涨幅最大** — 含 20cm 未封住的票、未涨停但大涨的百亿大票
2. **带动性最强** — 板块旗手地位，其他股是被它带动的跟风
3. **资金最猛攻** — 成交额 + 龙虎榜净买入方向 + 主力净流入综合最强

**Why**：涨停只是表面。真龙可能是科创板涨18%没封住的票，或百亿成交+10%的中盘真龙。默认"涨停=真龙"会把炒小炒妖的孤狼当主线（典型情境：某SiC板块A股涨停被误判为板块龙头，实际只是跟风票，真龙是当天未涨停但板块周期涨幅最大+带动性最强+资金最猛的另一只），错过真正的板块旗手。

**How to apply**（操作级）：
- 候选股纳入清单时，**强势股池里的百亿大涨真龙必须与涨停股并列评估**，不能只看涨停板
- 11维度评分里"龙头地位"维度（20分）按此三标准打分，**不能只看连板高度**
  - 板块周期涨幅最大 + 带动板块共振 + 资金最猛攻 = 20分（即使没涨停）
  - 仅涨停高板但板块无共振、资金不猛 = ≤10分
- 同一板块出现"涨停股"和"大涨未涨停的百亿股"时，按三标准综合判定真龙——**不要默认涨停股就是真龙**
- 涨停股可能只是"情绪龙"，真龙可能是百亿成交+大涨未涨停的板块旗手
- 与规则5嵌套：连板维度的"龙头/板块龙头"称谓仍按≥2/≥3板执行；强势股池真龙按三标准认定，称"百亿真龙/板块旗手"，**两个维度的称谓不能互相误用**

### 第一步：计算情绪温度（V5.1量化公式 — 焊入接力维度）

**⚠️ V5.1 升级**：旧公式只看当天截面（涨停数/上涨数/炸板率），漏掉了龙头选手的命根子——**接力赚钱效应**。一个市场涨停 80 家、但昨日涨停今天集体绿盘，对龙头选手就是冰点，旧公式却照样算出"高温"。V5.1 把接力维度焊进公式。

**情绪温度计算公式 V5.1**：
```
基础温度 = (涨停数/上涨数) × 连板高度系数 × (1-炸板率) × 100

接力修正系数 K（来自 premium_ladder.py 的 premium.level）：
- 强赚钱效应（昨日涨停今日均涨 ≥+3%）  → K = 1.2
- 弱赚钱效应（均涨 0 ~ +3%）          → K = 1.0
- 弱亏钱效应（均涨 -3% ~ 0）          → K = 0.8
- 强亏钱效应（均涨 <-3%）             → K = 0.6

情绪温度 = 基础温度 × K

温度分级：
- 冰点（<20）  ：只做超跌反弹，空仓为主
- 回暖（20-40）：可以低吸龙头，2-3成仓
- 高温（40-60）：追涨龙头，3-5成仓
- 过热（>60）  ：谨慎，随时准备撤退，只持有已盈利票

关键指标：
- 涨停数：>50个=活跃，30-50=正常，<30=低迷
- 连板高度：≥5板=情绪高潮，3-4板=健康，≤2板=分化
- 炸板率：<30%=健康，30-50%=分化，>50%=情绪转弱
- 赚钱效应：昨日涨停今日均涨幅+红盘率（接力是否赚钱，龙头选手第一指标）
- 首板晋级率：>40%=接力意愿强，20-40%=正常，<20%=接力断档
- 连板晋级率：昨日≥2板今日继续涨停比例，<30%=高度板危险
```

**🚦 群龙无首判断（V5.1 新增 — 龙头选手最该空仓的信号）**

情绪温度算完后，必须再下一个独立结论：**今日市场龙头辨识度**。对只做龙头的选手，没有明确总龙头本身就是不参与信号——评分再高的候选，群龙无首时也不值得开仓。

| 辨识度 | 判定标准（涨停梯队 + 板块共振交叉验证） | 结论 |
|--------|----------------------------------------|------|
| ✅ 明确单龙 | 最高板独一档 + 所属板块涨停≥3只共振 + 成交资金最猛 | 有龙可做 |
| 🟡 双龙卡位 | 两只票同处最高/次高板抢龙头地位，板块各自有共振 | 等卡位战分出胜负 |
| 🚨 群龙无首 | 最高板独苗且板块无共振 / 多个3-4板分属不同板块互不带动 / 梯队断层严重 | 龙头选手空仓信号 |

> 群龙无首时，本步必须明确写一句"今日无明确总龙头，龙头选手不参与"，并在市场概况结论里置顶。

**动态仓位管理（根据情绪温度）**：
```

- 总仓位：0-1成
- 持股数：0-1只
- 策略：空仓为主，或1成仓试探超跌反弹

【回暖期 20-40】
- 总仓位：2-3成
- 持股数：1只

【高温期 40-60】

【过热期 >60】

- 持股数：1-2只（已持有的）
- 策略：随时准备撤退，落袋为安
```

### 第二步：逻辑强度评分（V5.0结构化评分）

**逻辑强度评分系统（满分100分）**：
```
逻辑强度 = 政策催化(30分) + 事件驱动(25分) + 业绩支撑(20分) + 
          题材热度(15分) + 机构认可度(10分)

评分细则：
【政策催化 30分】
- 中央级文件/会议：30分（如中央经济工作会议）
- 部委政策/规划：20分（如工信部产业政策）
- 地方政策/试点：10分（如地方补贴政策）
- 无政策支撑：0分

【事件驱动 25分】
- 突发重大事件：25分（如地缘冲突、自然灾害）
- 预期内事件：15分（如财报、新品发布）
- 周期性事件：5分（如节假日消费）
- 无事件驱动：0分

【业绩支撑 20分】
- 业绩暴增（>100%）：20分
```

**🔑 生命周期判断必须用第零步「板块轮动强度数据」交叉验证（platerotat.py）**

第零步第 5 项已拉取 `plate_trends`，每个板块的 `ranks`（近N日强度排名走势）+

### 第四·五步：滚动型大盘股板块的特殊处理（V5.1 新增 — 防止误用科技小盘股规则）

**识别**：当复盘对象（龙头候选/重点跟踪标的）属于以下板块时，触发**滚动型规则**：
- 电力、银行、证券、煤炭、有色金属、航运、石油、钢铁、保险、白酒、地产央企、
  中字头央企（中国XX系列）、大型蓝筹

**为什么需要特殊处理**：电力这类大盘股板块的节奏与科技小盘股截然不同——它们是**滚动慢牛**（个股轮动 + 大资金分批进场），不是"集中爆发→一波到位"。把科技股的"跟风密度阈值/炸板容忍度/巨量解读"硬套到电力，会反复误判（5-08 大唐发电就是这么打脸的）。

**特征对比（板块周期判定 + 评分时套用）**：

| 维度 | 科技小盘股板块（默认规则） | 滚动型大盘股板块（特殊规则） |
|---|---|---|
| 跟风密度阈值 | ≥3 只涨停共振 | **≥2 只大票(流通≥200亿)即算共振** |
| 炸板容忍度 | ≥5次=警示(走 HARD GATE 1.6) | **≥10次仍可能是机构换手**（5问 Q2 量能换手不扣分） |
| 板块周期 | 集中爆发，一波到位 | 滚动慢牛，可拉长数周 |
| 板块衰退判定 | 跟风密度 > platerotat | **platerotat 强度榜 > 跟风密度**（platerotat 不可用时用涨停密度证据） |
| 巨量解读 | 巨量=危险信号 | **巨量=机构分批进场，常态** |
| 11 维度·龙头地位打分 | 按跟风密度+连板高度 | **加权倾向大票共振，2 只大票即可给 ≥15 分** |

**判定流程**：
1. 第零步数据查询时，对每只 ≥2 板个股标注"是否滚动型板块"（电力/银行/煤炭等名册比对）
2. 是滚动型 → 跟风密度阈值放宽（≥2 只大票即算共振）+ 炸板容忍放宽（≥10 次仍可能是分歧日）
3. 板块周期判定优先看 platerotat；platerotat 不可用时用涨停密度（板块涨停只数 ≥3 即算共振）
4. 11 维度评分「龙头地位」按滚动型权重打分（参见上表）

**报告里必须标注**：滚动型板块龙头的体检表 + 评分表里，加一行「✅ 滚动型大盘股板块·规则放宽」标识，让用户知道用的是哪套规则。

### 第五步：龙头分类（V3.1周期理论 + V5.0量化优化）

**🔥 必须先做：周期定位分析（基于真实数据！）**

在给龙头分类之前，必须先完成以下数据查询和分析：

**步骤1：查询当前龙头完整数据**
- 使用"第零步"中的查询命令，获取当前龙头的完整连板周期
- 确认首板日期、是否炸板、最高连板数

**步骤2：查询同板块历史龙头**
- 使用"第零步"中的查询命令，找出同板块其他龙头股票
- 对比时间线：其他龙头的首板日期、开板日期

**步骤3：周期判断（基于数据对比）**
根据查询到的真实数据，判断当前股票属于哪种类型：

1. **首波龙头判断标准**：
   - 同板块没有其他龙头在它之前启动
   - 或者同板块之前的龙头已经沉寂很久（>30天）
   
2. **补涨龙判断标准**：
   - 同板块存在首波龙头
   - 首波龙头开板日期 vs 当前股票首板日期：时间差在±1-2天内
   - 跟风股少（<3只）
   
3. **二波龙判断标准**：
   - 就是首波龙头本身
   - 回调一段时间后（10-30天）重新启动连板

**⚠️ 重要：必须用真实数据验证，不能凭记忆判断！**

---

#### 类型A：首波龙头（板块启动的第一个龙头）
**特征**：
- 板块启动时最早涨停的龙头
- 题材：AI、机器人、华为链、新能源、燃气等（持续性强）
- 连板高度：首波10-18连板
- 跟风股：多（>5只）
- 板块持续上涨

**识别方法**：
- 同板块没有其他龙头在它之前启动
- 或者同板块之前的龙头已经沉寂很久（>30天）

**操作（重点！）**：
- **首波连板期**：重仓30-50%，吃连板爆发力
- **连板结束日**：减仓至20%（不是清仓！），加入"二波监控池"
- **回调期**：保持20%仓位，每日观察题材热度
- **二波连板启动**：重新加仓至30-50%，再吃一轮连板
- ⚠️ 如果回调超过30天且题材冷却：减至10%或清仓

**为什么不清仓？**
- 历史上多次出现首波连板结束后清仓删自选，结果错过同一只票数月后的二波连板行情
- 保持20%仓位的意义：既不会踏空二波，又不会在回调期亏太多

**案例（句式模板，禁列个股）**：某AI板块首波龙头跨年走出"15连板→回调→18连板二波"的双波行情

---

#### 类型B：补涨龙（板块末期的接力龙头）⚠️
**特征**：
- 首波龙头开板当天或次日启动
- 同板块、同题材
- 跟风股：少（<3只），因为板块资金已经疲软
- 连板高度：通常5-8板就结束（不会像首波龙头那样10+板）

**识别方法（关键！）**：
1. 使用"第零步"的查询命令，获取同板块历史龙头数据
2. 时间线对比：首波龙头开板日期 vs 当前股票首板日期
3. 如果时间吻合（±1-2天）→ 大概率是补涨龙
4. 如果跟风股少（<3只）→ 进一步确认是补涨龙
5. **必须验证板块一致性**：补涨龙必须与首波龙头属于同一板块

**⚠️ 常见错误案例（句式模板，禁列个股）**：
- ❌ 错误：某电池板块龙头4月18日开板 → 某燃气板块新龙头4月20日首板 → 判断为补涨龙（实际两个板块不同，不构成补涨关系）
- ✅ 正确：板块不同就不能判断为补涨龙关系
- ✅ 正确：上例中的燃气新龙头应判断为燃气板块自身的首波龙头（如4月21日炸板则定性为质量较差的首波）

**操作（重点！）**：
- **连板期**：轻仓参与10-20%（不要重仓！）
- **连板结束日**：立刻清仓，不加入监控池
- ❌ **不要幻想二波**：补涨龙是板块末期，二波概率极低
- ❌ **不要等回调**：连板结束就是板块周期结束

**为什么不等二波？**
- 补涨龙出现时，板块资金已经从首波龙头撤出
- 板块周期已经走完，资金会切换到新题材
- 补涨龙的二波概率<10%，不值得等待

**案例**：需要基于真实数据验证后补充（暂无确认案例）

---

#### 类型C：二波龙（首波龙头回调后的再启动）
**特征**：
- 首波龙头回调一段时间后（通常10-30天）
- 题材出现新利好或市场情绪回暖
- 重新启动连板

**识别方法**：
- 就是首波龙头本身，或者同板块新龙头
- 回调后缩量企稳，然后放量突破

**操作**：
- 二波启动确认 → 重仓30-50%
- 二波连板结束 → 减仓至20%，可以等三波（如果题材够强）
- 二波失败（跌破首波启动价）→ 清仓止损

**案例（句式模板，禁列个股）**：某AI板块龙头跨年走出"15连板首波 → 回调 → 18连板二波"

---

#### 类型D：纯情绪妖股（快进快出）
**特征**：
- 题材：重组、跨界、壳资源、控制权变更
- 连板高度：10-16连板
- 跟风股：少（<3只）
- 无板块逻辑，纯情绪炒作

**操作**：
- 首板、二板可以打
- 连板结束当天或次日必须清仓
- ❌ 不要幻想二波，连板结束立刻删自选

---

#### 类型E：先锋龙（大行情信号）
**特征**：
- 连板高度：6-8连板（不高）
- 出现时机：市场冰点后的第一个龙头
- 关键信号：看它能否带动跟风股持续涨停

**操作**：
- 小仓位参与（10-20%），主要用来观察市场反应
- 重点看后续3-5天：
  - 如果同题材出现3只以上跟风涨停股 → 说明主线来了，加仓
  - 如果只有它一个涨停，无跟风 → 假突破，清仓观望
- 如果主线确认，立刻切换到"正宗龙头"

### 第六步：理想标的11维度评分（V5.0量化系统）

**评分维度与权重**：
```
总分 = 核心维度(50分) + 重要维度(30分) + 辅助维度(20分)

【核心维度 50分】
1. 逻辑强度（15分）：≥70分=15分，50-69分=10分，<50分=5分
2. 主动性（15分）：≥70分=15分，50-69分=10分，<50分=5分
3. 龙头地位（20分）：
   - 板块龙1：20分
   - 细分龙1：15分
   - 板块前3：10分
   - 板块前5：5分

【重要维度 30分】
4. 唯一性（10分）：唯一=10分，2只=7分，3-5只=4分，>5只=2分
5. 带动情绪（10分）：
   - 带动板块启动：10分
   - 板块内最强：7分
   - 跟随板块：4分
   - 独立行情：2分
6. 大众情人（10分）：
   - 多个游资席位：10分
   - 单一游资+机构：7分
   - 纯游资或纯机构：4分
   - 无明显主力：0分

【辅助维度 20分】
7. 成交容量（3分）：>10亿=3分，5-10亿=2分，<5亿=1分
8. 上涨趋势（3分）：20日线上=3分，60日线上=2分，其他=0分
9. 跟风股（3分）：有3只以上=3分，1-2只=2分，无=0分
10. 未加速（3分）：未连续3天缩量=3分，已加速=0分
```

**止损线（严格执行）**：
```
【当日止损】
- 炸板不回封：盘中立即止损（除非是洗盘性质的秒开秒封）
- 尾盘开板：14:30后开板，盘中止损
- 大幅开板：开板后跌幅>3%，盘中止损

【次日止损】
- 竞价低开>3%：竞价出
- 开盘后快速跳水：跌破昨日均价线，盘中止损
- 板块龙头换人：同题材出现更强标的，盘中换股

【情绪止损】
- 市场情绪转冰点（<20）：全部清仓
- 连板高度断层（5板直接到2板）：减仓一半
- 炸板率>50%：不再新开仓，持有的设好止损
```

**止盈线（分批止盈）**：
```
【首板止盈】
- 不设止盈，看次日表现

【2-3板止盈】
- 不设止盈，看情绪和板块
- 如果板块开始分化，减半仓

【4-5板止盈】
- 减半仓，防止分化
- 如果次日继续强势，可以加回来

【6板以上止盈】
- 只留底仓（1-2成），落袋为安
- 如果冲击10板，可以留底仓博弈

【特殊止盈】
- 单日涨幅>15%（未封板）：尾盘减半仓
- 连续3天缩量加速：减半仓（加速末期）
- 暴量涨停（换手>30%）：次日冲高减半仓
```

**换股策略**：
```
【主动换股】
- 持有的票走弱，同题材出现更强的 → 盘中换股
- 持有的票已经高位（5板以上），新龙头刚启动 → 减仓换股
- 板块生命周期进入衰退期，新板块启动 → 换板块

【被动换股】
- 持有的票炸板，同题材龙头还在强势 → 止损后换龙头
- 持有的票逻辑证伪，新逻辑出现 → 止损后换新逻辑

【换股原则】
- 不要频繁换股（一天最多换1次）
- 换股要换更强的（评分更高、主动性更强）
- 换股要在盘中完成（不要隔日换）
```

## ⚠️ 重要提醒（V5.0核心升级）

### V5.0 vs V4.0 核心区别

**V4.0（旧版）**：
- 5维度分析（人气股、资金流入、涨停股、龙虎榜、板块龙头）
- 情绪温度：龙头高度+涨停家数+炸板率+指数走势（100分制）
- 打分体系：10分制，≥7分买入
- 缺少退出机制

**V5.0（新版）**：
- ✅ **7源股票池**：增加开盘啦多日叠加、异动提醒
- ✅ **情绪温度计**：新公式 = (涨停数/上涨数) × 连板高度系数 × (1-炸板率)
- ✅ **逻辑强度量化**：100分制（政策30+事件25+业绩20+热度15+机构10）
- ✅ **主动性量化**：100分制（封板速度30+封单强度25+分时形态20+换手15+大单10）
- ✅ **11维度评分**：100分制，≥80分强烈买入，70-79买入，60-69观察
- ✅ **板块生命周期**：启动期/加速期/分化期/衰退期自动识别
- ✅ **动态仓位管理**：根据情绪温度自动调整（冰点0-1成、回暖2-3成、高温3-5成、过热只持有）
- ✅ **完整退出机制**：止损线+止盈线+换股策略
- ✅ **复盘日志模板**：每日/每周/每月结构化记录

### 龙头分类决定操作（V3.1周期理论）
1. **首波龙头**：连板结束减至20%，等二波连板
2. **补涨龙**：连板结束立刻清仓，不等二波（板块末期）⚠️
3. **二波龙**：重仓参与，往往比首波更猛
4. **妖股**：连板结束就跑，不等二波
5. **先锋龙**：观察跟风，决定是否切换到正宗龙头

### 周期理论是关键（V3.1）
- **时间线对比**：首波龙头开板日 vs 当前股票首板日
- **补涨龙识别**：首波龙头开板当天启动的同板块股票 → 大概率是补涨龙
- **补涨龙特征**：跟风少、连板高度低（5-8板）、二波概率<10%
- **⚠️ 判断依据**：必须基于"第零步"查询的真实数据，包括龙头完整连板周期、同板块历史龙头时间线

### 二波龙监控是关键
- 历史经典：某AI板块龙头跨年走出"15连板首波→回调→18连板二波"
- 如果首波连板结束就清仓，就错过了二波的18连板
- 保持20%仓位的意义：既吃到首波，又不踏空二波
- ⚠️ 但补涨龙例外：补涨龙连板结束必须清仓，不等二波

### 情绪温度决定仓位（V5.0）
- 冰点期（<20分）：空仓为主，0-1成仓
- 回暖期（20-40分）：低吸龙头，2-3成仓
- 高温期（40-60分）：追涨龙头，3-5成仓
- 过热期（>60分）：只持有已盈利票，不新开仓

### 从最强变弱，立即离场
- 龙头炸板当天，减仓50%
- 次日不封板，全部清仓
- 不幻想、不预测、不死扛

### 买入原则（V5.0）
1. 最多只能买2只（多了看不过来）
2. 单只最多半仓（仓位太重影响心态）
3. 做完一个，再做另一个
4. 优先买评分≥80分的
5. 如果仓位乱了，停止任何买入

### 铁律（必须遵守）
1. 仓位要配置好，如果乱了，停止任何买入
2. 做完一个，再做另一个（不要同时操作多只）
3. 行情不是1天走完的，但是钱可以被亏完
4. 尽量自己去亲力亲为地看（看K线、看板块、看分时）
5. 严格执行止损，不要心存侥幸
6. 不要追高，宁可错过，不要做错
7. 不要频繁交易，每天最多操作2次
8. 不要在情绪冰点期重仓，不要在情绪过热期追高
