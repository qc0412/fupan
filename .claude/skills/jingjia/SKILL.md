## 🎯 集合竞价的特殊性（必须先理解）

集合竞价不是连续竞价，它的资金行为有 3 个独特特征：

1. **两阶段意图差异巨大**：
   - **9:15-9:20**：可挂可撤 → **试盘期**（主力挂大单试盘后撤掉是常见操作，意图最暧昧）
   - **9:20-9:25**：只能挂不能撤 → **真金白银期**（这 5 分钟挂的单必须成交，意图最可信）
   - 9:25 集中撮合，产生开盘价
2. **没有"实时涨幅"，只有"竞价涨幅"**：竞价涨幅 = 9:25 撮合价 vs 昨日收盘价
3. **竞价成交额很小**：通常只有全天成交的 1%-5%，所以净流入绝对值不能跟盘中比，**占比远比绝对值重要**

## 🩸 核心血泪教训（每次必读，五条都曾让判定崩盘）

1. **一字涨停 ≠ 可买**。竞价就涨停（涨幅 ≥ +9.8%）的票直接排除候选池，但必须**标注为参考龙头**，
   它是判断"板块主线方向"的锚点。打板党想接力只能盯次日，竞价当天买不到。
2. **高开诱多陷阱**。竞价 +5% ~ +9% 高开 + 大净流入，看似强势 = **9:30 开盘冲高即砸**的概率高于真启动。
   主力高开是为了"出货"还是"启动"？看竞价撤单率（9:20 后是否在加单）+ 板块共振 +1 分钟分时。
3. **低开+大流入 99% 是出货陷阱**。竞价 ≤ -3% 低开却净流入排前列 = 主力低位**接出货盘**，不是抄底。
   真主力抄底从不在竞价就暴露意图，会盘中分批吸。
4. **平开/微高开+大净流入+占比高 = 真正的埋伏**。这是竞价维度最值钱的形态：
   主力刚动手、没拉抬、但已经下决心。需配合板块共振判定才能成立。
5. **板块独苗的竞价强势 = 题材独行**。候选票必须和当日**竞价就封板/接近封板**的票同板块或紧密关联。
   独苗强势（板块无共振）属高风险妖股，不进推荐池。

## 四条铁律

1. **判定主体只看"可买入的票"**——竞价就涨停的排除，但留作参考龙头。
2. **必须算"净流入占竞价成交比"**——竞价成交盘子小，占比比绝对值重要 10 倍。
3. **必须做板块共振判定**——候选票必须和当日竞价封板龙头同板块（或紧密关联），独苗强势直接降级。
4. **跨日跟踪资金力度变化**——如果有前一日数据，必须对比同一只票的资金力度变化，衰减 = 出货警示。

## 流程（5 步，缺一不可）

> ⚡ **提速分工（2026-06-12）**：第一、二、三步（含三步·补）和第五步的**机械判定已由脚本逐行预算**（见"脚本预判"列），以下条文是判定标准的定义与依据，供你**校验脚本结果、处理边界情况**用，不要从零重算。你的增量工作集中在：第四步板块共振软校验、D 级的"龙头/基本面"人工门槛、同级决胜与定稿。

### 第一步 · 扫榜识别"参考龙头"
- 找出所有**竞价涨幅 ≥ +9.8%**（一字涨停）或**+7% ~ +9.8%**（高开接近涨停）的票
- 标注它们的板块属性 = **当日主线方向**
- 这些票**排除出候选池**（一字买不到/高开追不上），但作为"板块共振"的判定锚点

### 第二步 · 三态 × 净流入分类

竞价没有"盘中走势"，只有**开盘那一刻的高/平/低开三态**，所以分类维度是：

```
                   净流入大
                       │
   ⚠️ 低开接盘陷阱      │  ⚠️ 高开诱多陷阱      │  🎯 弱转强候选（主推）
   (≤ -3% + 大流入)     │  (+5%~+9% + 大流入)    │  (-2%~+3% + 大流入 + 占比高)
   → 99% 出货           │  → 高开冲高即砸的概率高 │  → 主力刚动手已下决心
                       │                       │
   ─────────────────────┼───────────────────────┼─────────────────────  净流入中等/小
                       │                       │
   💀 弱开弱势          │  💤 高开跟风          │  ⚪ 平开观望
   (≤ -3% + 小流入)     │  (+5%~+9% + 小流入)    │  (-2%~+3% + 小流入)
   → 拉黑               │  → 一日游概率高        │  → 不参与
                       │                       │
                   净流入小
        ←  低开 (≤ -3%)    平开/微动 (-2%~+3%)    高开 (+5%~+9%)  →
```

注意：**+3% ~ +5% 是模糊带**，归"微高开"，配合占比和板块共振判定。

每只票必须落到一个格子里。

### 第三步 · 计算"净流入占竞价成交比"= 资金坚决度

```
净流入占比 = 竞价净流入 / 竞价成交额
```

| 占比 | 判定 |
|---|---|
| ≥ 70% | 🔥 资金极度坚决（几乎没人卖、撮合的全是主动买） |
| 50% ~ 70% | ✅ 坚决（主力主导） |
| 30% ~ 50% | ⚪ 中等（多空互有承接） |
| < 30% | ⚠️ 分散（资金不集中，警惕对倒/搬砖）|

**注意**：竞价占比阈值比盘中高（因竞价撮合方式是单次集中，占比天然偏极端）。
**这个指标比绝对值更重要 10 倍**——竞价成交盘子小，500 万净流入占比 80% 远比 5000 万净流入占比 5% 值钱。

> 净额占比 ≥45% = 资金坚决（B级以上）。
> - **网站「竞价净额」tab**：≥45% 整行高亮标注。
> - **md 报告**：单列一节「净额占比 ≥45%（资金坚决，全榜）」，把这几只都列出来，用**表格**（列：票名(代码)/占比/竞价涨幅/换手），**不带 🔥**。其余行不标注。⚠️ 不要写成斜杠分隔的 bullet 列表（渲染到网页是挤成一团的散文，很丑）。

#### 第三步·补 · 看"竞价换手%"= 竞价资金相对个股自身体量的力度（跨个股可比）

**口径（2026-06-08 改）**：直接用竞价数据自带的 `竞价换手%`（= 竞价成交额 / 流通股本），`fetch_jjyd.py` 已输出该列，**不再外部取昨日成交额**（短线侠竞价数据里本就有，无需东财日K，避免依赖外网）。

```
竞价换手% = 竞价成交额 / 流通股本    ← 这笔竞价相对个股盘子算不算"大动作"（数据自带）
```

| 竞价换手% | 判定 |
|---|---|
| < 0.2% | ⚪ 常态（竞价没用力，多为大盘子毛毛雨）|
| 0.2% ~ 0.5% | 偏活跃 |
| 0.5% ~ 1.5% | 🔥 竞价显著放量，资金真用了力（重点盯）|
| > 1.5% | ⚠️ 极端，多为小盘抢筹/异动，谨防买不到 |

> ⚠️ 上表阈值是按 `竞价换手%`(成交/流通股本) 量纲**初标**的（2026-06-08 实测：大盘权重票 0.08%~0.3%、小盘真用力票 1.5%+）。旧版"竞价成交/昨日全天成交额"口径的 2%/4%/8% 阈值**已作废**；D 级换庄门槛已按新口径重定为 ≥1.0%、跨级决胜"没用力"线 <0.3%（2026-06-12，与脚本 `D_TURNOVER_MIN`/`WEAK_TURNOVER` 常量同源，样本少属初定、可按实战回测微调）。

**为什么必须看这一刀**（血泪点，2026-06-01 实证）：
**绝对净额排名会骗人**——盘子大的权重票（如中际旭创流通 9534 亿、立讯 3001 亿），竞价随手挂挂就有几千万~上亿净额、能排进前 5，但竞价换手仅 0.08%~0.14%，**主力根本没用力**。反之小盘的豫能（流通 114 亿）竞价换手 1.5%，才是真在"用力换手"。
**口径**：竞价换手高 ≠ 能买，只代表"这笔竞价是真异动、值得盯"；方向（建仓 / 出货）仍需结合涨幅形态 + 占比 + 后续走势定性。**同榜跨股决胜以竞价换手% 相对高低为准**，不要用净额绝对值排序。

### 第四步 · 结构化分级筛选（资金维度 P1，板块共振 P2 软校验）

**核心原则**：资金维度（占比 + 净流入 + 涨幅形态）是 P1，板块共振是 P2 软校验，不是 HARD GATE。
独苗 + 资金极坚决 = 主力**提前埋伏的早期信号**，独苗不能一票否决。

**先执行第五步排除清单**，再在剩余候选里按以下分级筛选：

| 级别 | 占比阈值 | 涨幅形态 | 换手烈度 | 判定 |
|---|---|---|---|---|
| **A 级（最强档）** | ≥ 60% | 平开/微高开 (-2%~+3%) | — | 🥇 主推 |
| **B 级（坚决档）** | ≥ 40% | 平开/微高开 (-2%~+3%) | — | 🥇 主推（A 级无人时）|
| **C 级（合格档）** | ≥ 30% | 平开/微高开 (-2%~+3%) | — | 🥇 主推（B 级无人时）|
| **D 级（换庄候选）** | ≥ 40% | **深水低开 (≤ -7%)** | **竞价换手 ≥ 1.0%** | 🥇 可进主推，但必带"低开巨震"标注，见下 |

**⚠️ D 级·换庄候选（2026-06-01 风华高科血泪，专为"深水低开但主力真用力"开的口子）**：
A/B/C 级把"低开"整个判进陷阱区，但实战里**深水低开 + 高换手 + 龙头/基本面**恰恰是新主力建仓最早的脚印（见第五步分流），它资金力度往往**碾压平开候选**，不该被闸门摁掉。准入门槛（四条全中才算 D 级）：
1. 深水低开 **≤ -7%**（浅水 -3%~-7% 不够，倒货嫌疑大）
2. 竞价换手 **≥ 1.0%**（新口径=竞价成交/流通股本；落在"0.5~1.5% 真用力"档上半段 = 主力相对个股体量真用了力）
3. 净流入占竞价成交比 **≥ 40%**（买盘主动、非对倒）
4. **标的是龙头 / 有基本面**（独苗题材不进 D 级，留在第五步警惕区）

D 级进推荐时**必须钉死三条标注**，不许包装成干净弱转强：
- 「**低开巨震票，非确定性单**」——买点是**次日/盘中回踩不破前低的弱转强**（走 /zhuanqiang），不是竞价打；
- 「**换庄未证伪**」——要龙虎榜接货席位（机构/一线游资=含金量高）+ 次日缩量承接来签字；
- 「**轻仓试错**」——高赔率需验证，不是重仓确定性买入。

**同级内决胜**：**竞价换手% 更高者优先**，不要再用净流入绝对值排序——绝对值会被大盘子票刷出来骗人（2026-06-01 实证：中际旭创净 4728 万排前 5，但竞价换手仅 0.1%，是常态毛毛雨）。竞价换手才是"主力相对个股体量真用了多大力"的可比标尺。
**跨级决胜（关键）**：当平开 A/B/C 级候选**竞价换手全 < 0.3%（主力没用力）**、而场上有合规 D 级（竞价换手 ≥ 1.0%）时，**D 级优先占主推位**——宁推"主力真下注但需验证"，不推"形态干净但主力没动手"。（脚本预筛小结遇到该情形会输出"⚡ 跨级决胜提示"）

**板块共振作为软参考**：
- 同板块 + 当日参考龙头一字封板 → 候选评级加成
- 紧密关联板块 → 候选评级加成
- 板块独苗 → 不降级（资金极强时反而是埋伏信号）

⚠️ 当板块归属拿不准时，**调用 plate_classifier.py 工具**（`~/.claude/skills/fupan/scripts/plate_classifier.py --classify CODE`），不要凭东财行业字段拍脑袋归类（参考 /jieli 铁律#3）。

### 第五步 · 陷阱识别（排除清单）

逐只过滤掉以下"伪强势"：

| 陷阱类型 | 特征 | 判定 |
|---|---|---|
| **一字涨停** | 竞价涨幅 ≥ +9.8% | ⭐ 排除候选，标参考龙头 |
| **高开诱多陷阱** | 竞价 +5% ~ +9% + 净流入大（未封板） | ⚠️ 排除，开盘冲高即砸概率高 |
| **低开接盘陷阱** | 竞价 **-3% ~ -7%**（浅水低开）+ 净流入排前列 + 占比中等 | ⚠️ 警惕倒货，但**不许机械判死**，须看下面"换庄"分流 |
| **对倒嫌疑** | 净流入占比 < 30% + 竞价成交额异常大 | ⚠️ 警惕，可能是搬砖 |
| **板块独苗** | 同板块无竞价封板共振 + 净流入大 | 🎲 降级，妖股射程 |
| **资金衰减** | 昨日竞价在榜+强势 → 今日净流入衰减+竞价涨幅转弱 | 💀 出货信号 |

> **⚠️ "低开接盘 vs 换庄承接"分流（2026-06-01 风华高科血泪，禁止再机械判死）**
> "低开 + 大净流入"是**中性信号**，倒货和坚决拿货长得一模一样，单凭竞价无法定性。判死前必须分流：
> - **深水低开（≤ -7%）+ 大额主动净买 + 竞价换手高（≥ 1.0%，新口径=竞价成交/流通股本）+ 标的是龙头/有基本面** → **更可能是换庄/坚决承接**，不是倒货。逻辑：地板价没有获利盘可骗、散户也不会在 -7% 接，花真金白银主动扫货做局收益为负，没人这么干。**四条全中即升 D 级·换庄候选，可进主推**（见第四步），但必钉"低开巨震/换庄未证伪/轻仓试错"三标注——买点走次日盘中回踩弱转强（/zhuanqiang），不是竞价打。
> - **浅水低开（-3% ~ -7%）+ 占比中等 + 题材独苗/无基本面** → 倒货概率高，维持 ⚠️ 警惕。
> - 共同点：**"是不是新主力"竞价当下不可证伪**，要等龙虎榜接货席位（机构/一线游资=换庄含金量高）+ 次日是否缩量承接来签字。
> - A 股**最贵的陷阱都是花大钱做出来的**（高位对倒/诱多，钱靠倒货赚回来）——所以"花钱多"不证明不是陷阱，**关键看价格位置**：高位花钱拉抬=陷阱，深水位花钱扫货=难做局的真承接。

**核心理念**：集合竞价是**主力意图最早的公开声明**——比盘中走势更早，比消息面更直接。
最强的票不是竞价涨得最多的（已经封死了），也不是竞价流入最大的（可能是低开倒货）；
**而是"主力刚动手、还没拉、但已经下决心"的那只票** —— 平开/微高开 + 大净流入 + 占比高 + 板块共振。

**避错心法**：
- 一字涨停 = 板块锚点（买不到，只参考）
- 高开 +5%以上 + 大流入 = **诱多陷阱**（开盘冲高即砸概率 > 启动）
- 低开 -3%以下 + 大流入 = **出货陷阱**（99%）
- 平开/微高开 + 大流入 + 占比 ≥ 50% + 板块共振 = **真正的埋伏机会** ✅
