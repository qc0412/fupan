- **砍掉**：方法论复述、免责声明、教学铺陈、大段推理、重复解释、抒情判断。
- **保留**：
  1. **每一只 2板及以上**个股都要分析；

  3. 单独汇总**每个板块的龙头**：只列板块名、龙头、跟风数、结论；

## 这个 skill 只做一件事

盘后把当日所有**连板股（2板+）**拉出来，逐只判定**「它到底是不是龙头」**。
**判定主体是连板股，但判定的证据必须扫到首板**——首板是连板龙头「跟风」的主要载体。
产出【接力盯盘池】：哪些连板是真龙头（次日值得盯接力）、哪些只是跟风 / 补涨龙 / 独苗妖股。

> **定位**：只建池 / 观察，**不实盘下单**。实盘买点仍只有 `/zhuanqiang` 的弱转强。
> `/jieli` 回答「**谁是龙头**」，`/zhuanqiang` 回答「**龙头什么时候能买**」。
> ⚠️ `/zhuanqiang` 是规划中的 skill，**尚未实装**——此处仅作职责边界声明（买点不归本 skill），不要尝试调用它。

## 🩸 核心血泪教训（每次必读，四条都曾让判定崩盘）

1. **连板高 ≠ 龙头。** 独苗连板再高也只是妖股——真龙头的命根子是**带动性**。
2. **带动性不是拍脑袋打钩——必须实数跟风。** 把板块里的「连板跟风 + 首板跟风」
   **逐只列名数清楚**。没有跟风的连板 = 独苗妖股；**首板跟风在扩散 = 板块在长身体
   = 龙头带动性的硬证据**。漏扫首板，带动性这一项就是瞎打分。
3. **板块归属分错 = 全盘皆输。** 一只票常有多个概念，**东财行业字段经常把同板块
   的票打散到不同行业**——典型：大众交通挂「铁路公路」、德赛西威挂「汽车零部」，
   其实同属智能驾驶。**分错板块 → 跟风数错 → 带动性误判 → 龙头判定整个崩。**
   每只连板股必须列全所有概念、逐个对比，才能归对主板块。
4. **分歧日 ≠ 破坏日——炸板≥5次但回封了的连板，多半是真龙头的"分歧转一致"经典走势，不是衰退。**
   真龙头的命门不在"炸不炸"，而在"**炸后回不回封 + 次日缩不缩量**"。把"巨量+炸板+但回封"
   的分歧日一刀切判"破坏/衰退/排除"，等于错杀洗盘日的真龙。**典型反例**：2026-05-08 大唐发电
   3板炸11次但最终封死成3板，当时被判"补涨末端·电力衰退·🚫排除"；5月11日4板封死+板块4只
   涨停爆发，整个判定方向打脸。教训：分歧日必须走"⏸️待次日验证"灰度档，**禁止单日下死结论**。

## 四条铁律

1. **判定主体只看 2板+**；但**判定证据必须扫全部首板**（首板 = 跟风载体）。
   1板不是判定主体，但绝不能不看——它是连板龙头带动性的证据。
2. **连板 ≠ 龙头**。必须实数跟风，**独苗（0跟风）无论连板多高都判独苗妖股**。
3. **板块退潮期的连板，接力价值一律归零**（退潮诱多反抽，不是接力机会）。
   但「退潮」判定**禁止仅凭单日跟风弱**——必须叠加 platerotat 板块强度榜走势交叉验证
   （板块仍在 Top10 且 trend ≠ 下降 → 禁止判退潮）。
4. **炸板 ≥ 5 次的连板股，必须走「分歧日 vs 破坏日 5 问」识别**（见下方流程第 4.5 步），
   分歧日走 ⏸️ 待次日验证档，不进核心池但也不一刀切排除；破坏日才能判 🚫 排除。

## 流程（6 步，缺一不可）

### 第一步 · 圈定连板股（2板+）= 判定主体
`premium_ladder.py` 的 `ladder` 字段按高度逐档给。只取 `height ≥ 2` 作为待判定主体。

### 第二步 · 扫全市场涨停（含**全部首板**）= 判定龙头的证据底座
⚠️ **必须扫首板，不许省。** 理由：
- 首板是连板龙头「跟风」的主要载体——**没扫首板就数不清跟风**；
- 首板共振 / 扩散 = 板块在长身体 = 连板龙头带动性的硬证据；
- 漏扫首板 → 第四步带动性② 就是瞎打分。

把当日**全部涨停（连板 + 首板）按概念 / 板块归好堆**，作为第三、四步的底座。

### 第三步 · 逐只连板股「板块归属」HARD GATE —— 分错板块 = 全盘皆输

**强制工作流（先跑权威工具，再人工对比，禁止跳过工具直接拍脑袋归类）**：

**3.0 跑权威板块归类工具**（先做这一步，再做下面所有事）：
```bash
# 拿当日全部涨停股的权威板块绑定（选股通源，按"今日为什么涨"归类）
~/.claude/mcp-servers/astock-data/venv/bin/python3 \
  ~/.claude/skills/fupan/scripts/plate_classifier.py --surge --json > /tmp/jieli_surge.json

# 对全部连板股（2板+）反查权威板块
~/.claude/mcp-servers/astock-data/venv/bin/python3 \
  ~/.claude/skills/fupan/scripts/plate_classifier.py --classify CODE1,CODE2,... --json
```
**输出字段**：每只票拿到 `plates`（权威板块列表）+ `reason`（今日涨停原因原话）+ `board_count_desc`（"3天3板"等连板状态。注意：此为区间累计口径如"12天7板"≠连续连板数，禁止当连板数用；连板数一律以第一步 premium_ladder/AkShare 连板数（连续口径）为准）+ `source`（surge_stock=权威 / plate_set=反查 / N/A=未找到）。

**3.1 对每一只**连板股出一张归属矩阵：

```
🔍 [票名]([代码]) 板块归属
【权威板块归属】（plate_classifier --surge / --classify 结果，source=surge_stock 为权威）
  → 板块: ____ ｜ 涨停原因: "____"（接口原话）
【所有概念清单】（开盘啦 concept 字段补全，仅作 cross-check）
  ① 概念A   ② 概念B   ③ 概念C  …
【每个概念今日热度对比】
  | 概念 | 同概念连板数 | 同概念最高板 | 同概念首板数 | 成交TOP1 |
【主板块判定】优先采用权威板块；若权威板块与"概念热度对比"冲突，记差异、用权威，
              并在结论里写明"权威 vs 概念字段不一致：__"（这往往揭示新主线分化）
  ❌ 排除概念：____（理由：高度撑不起 / 密集度不够）
```

⚠️ 三个底线：
1. **禁止跳过 3.0 工具步骤**，直接靠概念/行业字段归类（违反铁律#3）。
2. 权威工具返回 `source=N/A`（未找到）的连板股，必须用 `--with-chain` 走 plate_set 反查；
   反查仍未命中，才允许走"概念字段+人工判定"兜底，但要标"⚠️ 工具未覆盖·人工归类存疑"。
3. 工具结果与开盘啦 concept 字段冲突时，**默认信工具**，并在体检表里记"权威 vs 概念字段不一致"。

### 第四步 · 逐只龙头体检（真龙三标准）
对每只连板股出体检表（见下），核心是【跟风清单·实数】+【真龙三标准】。

### 第 4.5 步 · 分歧日 vs 破坏日识别（炸板 ≥ 5 次的连板股强制走）

**血泪触发条件**：体检表 `炸板次数 ≥ 5` 时，**禁止**仅凭"炸板多+跟风弱"就判破坏/排除，
必须先回答下面 5 个问题：

| # | 问题 | 分歧日(健康洗盘) | 破坏日(真衰退) |
|---|---|---|---|
| Q1 | **最终是否回封？**（涨停状态=回封了，包括尾盘秒回） | ✓ 回封 | ✗ 未回封/尾盘开板 |
| Q2 | **量能是否异常放大？**（巨量但属换手 vs 巨量+杀跌） | ✓ 巨量(前N板均量×2+)但收盘=涨停价 | ⚠️ 巨量+收盘大幅低于均价 |
| Q3 | **板块强度榜位置**(platerotat) | ✓ 在 Top10 / trend 持稳或上升 | ✗ 跌出 Top10 / trend 下降 |
| Q4 | **同板块其他票是否跟跌？** | ✓ 板块其他票还在涨/不跌 | ✗ 板块集体熄火 / 跟风票跌停 |
| Q5 | **有无澄清/利空/出货明牌？** | ✓ 无 | ✗ 龙虎榜机构大额净卖>1亿 / 公告澄清 |

> Q5 反向同理：独苗票 + 机构大额净买 = 出货预警（见复盘心法·盛龙股份实证），不是定向接盘。

**判定规则**：
- 5 中 ≥ 3 个分歧日特征 → **⏸️ 分歧日·待次日验证**（走灰度档，不进核心池但也不排除）
- 5 中 ≥ 3 个破坏日特征 → **🚫 破坏日·排除**
- 双方均未达 3 或证据不全 → 默认走 ⏸️ 待验证（宁可等一天，不冒错杀风险）

⚠️ **此步是 HARD GATE**——炸板≥5次但跳过 5 问表直接判排除 = 违反铁律#4，必须重写判定。

### 第五步 · 接力价值分级
按「龙头判定 × 板块周期」查矩阵。

### 第六步 · 滚动型大盘股板块的特殊处理（识别+应用）

**识别**：当连板股属于以下板块时，触发**滚动型规则**：
- 电力、银行、证券、煤炭、有色金属、航运、石油、钢铁、保险、白酒、地产央企、
  中字头央企（中国XX系列）、大型蓝筹

**特征对比**：

| 维度 | 科技小盘股板块（默认规则） | 滚动型大盘股板块（特殊规则） |
|---|---|---|
| 跟风密度阈值 | ≥3 只共振 | **≥2 只大票(流通≥200亿)即算共振** |
| 炸板容忍度 | ≥5次=警示 | **≥10次仍可能是机构换手** |
| 板块周期 | 集中爆发，一波到位 | 滚动慢牛，可拉长数周 |
| 龙头判定优先级 | 跟风密度 > platerotat | **platerotat 强度榜 > 跟风密度** |
| 量能解读 | 巨量=危险信号 | **巨量=机构分批进场，常态** |

**判定流程**：识别为滚动型 → 跟风密度阈值放宽 → 炸板阈值放宽 → 板块周期以 platerotat 为准。

## 🧬 近两年情绪龙头样本启发（2023-2026，2026-06-09 新增）

> 研究材料：`/home/ubuntu/.openclaw/workspace/research/shortline_leaders_2023_2026.md`。/jieli 只做今日事实判定，但判定“谁像真龙头”时必须吸收这些历史样本规律。

### 1. 真龙头不是最高板，而是“高度 + 带动 + 主线级别”三合一

历史样本反复出现：鸿博/捷荣/常山/中文在线/克来/大众/双成/杭钢/利通/锋龙等，能成为阶段总龙头，通常不是因为单日最高，而是：
- **主线有强催化**：政策、全球技术爆点、爆款产品、产业涨价、并购重组、跨年情绪。
- **自身打开高度**：10cm 高度常见 8-15 板；20cm 则看连续大涨和区间涨幅。
- **能带动扩散**：连板跟风 + 首板跟风 + 趋势中军至少有一类持续响应。
- **分歧后能修复**：真龙头常有巨量/炸板/分歧日，但关键是回封、次日缩量或带动板块修复。

### 2. 龙头身份四分法（逐只 2板+ 必须归类）

| 类型 | 判定标准 | 接力含义 |
|---|---|---|
| 情绪总龙头 | 全市场最高辨识度；所在主线 S/A 级；跟风扩散或中军共振；高度打开赚钱效应 | 核心池，优先级最高 |
| 板块龙头 | 板块内最高或最主动；跟风数支撑；但全市场地位未必第一 | 只在板块未退潮时有价值 |
| 趋势中军 | 大成交/大市值/机构容量核心，不一定连板；验证主线容量 | /jieli 只作为证据，不放入连板主体池 |
| 补涨龙/分支龙 | 总龙头后排扩散、地域/字辈/同概念补涨，常高度不低但周期滞后 | 轻看，断板不等二波 |
| 独苗妖股 | 有高度但 0 跟风/无中军/单票公告驱动 | 标妖股，不按真龙接力 |
| 伪强势 | 高开诱多、大票净额毛毛雨、行业字段误分、蹭概念无扩散 | 排除或风险标注 |

### 3. 主线强度进入 /jieli 判定

逐只连板判定时，除“真龙三标准”外，必须给主板块做内部强度分级：
- **S级**：情绪总龙头 + 趋势中军 + 多层跟风同时存在；接近年度/月度主线。
- **A级**：强催化 + 多股涨停 + 连板梯队，可能走阶段主线。
- **B级**：有事件但扩散弱，只能支撑局部接力。
- **C级**：独苗、公告、蹭概念、无板块响应。

最终表格仍保持简洁，但“一句话证据”里要体现强度，如：`S级AI算力，首板5只+中军放量`、`C级公告独苗，0跟风`。

### 4. 催化权重校正

同样 2板，强催化主线优先级高于普通轮动：
1. 政策制度级/国家战略：新质生产力、低空经济、央企改革、并购重组、电力改革。
2. 全球技术爆点：AI、Sora、DeepSeek、人形机器人、芯片/算力。
3. 产业供需：涨价、短缺、国产替代、订单爆发。
4. 爆款消费/内容：华为 Mate、哪吒、短剧、旅游。
5. 情绪符号：跨年、生肖、字辈、地域。
6. 单票公告/传闻：除非带出板块，否则只判独苗/事件套利。

### 5. 每只 2板+ 的新增内部打分

在体检表内部增加“历史龙头相似度”，不必完整输出，但影响 `龙头判定`：
